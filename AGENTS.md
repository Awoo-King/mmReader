## Agent skills

### Issue tracker

Issues for this repo live as local markdown files under `.scratch/`. See `docs/agents/issue-tracker.md`.

### Triage labels

Triage roles use default canonical labels (`needs-triage`, `needs-info`, `ready-for-agent`, `ready-for-human`, `wontfix`). See `docs/agents/triage-labels.md`.

### Domain docs

This repo uses single-context domain docs: root `CONTEXT.md` and `docs/adr/`. See `docs/agents/domain.md`.
