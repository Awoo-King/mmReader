# mmReader

> 宇宙级漂浮 TXT / MD 阅读器。
>
> 作用很简单：把你那些字多得像诅咒、长得像天书、滚得像瀑布的文本文件，悬浮到 macOS 桌面上，优雅、离谱、近乎执念地读完它们。

如果你一直在寻找一种工具，能把 `.txt` 和 `.md` 像圣旨一样悬在桌面中央，顺手还能切状态栏、切 Dock、改快捷键、改文字颜色、记住窗口位置，甚至在你第二次打开时继续维持它那股“我早就不是普通阅读器了”的气势——恭喜，你已经找到了这位活祖宗。

## 这玩意儿到底会什么

- 会打开 `.txt` 和 `.md`，不挑文本，不装纯。
- 会按**视觉行**分页，不再按 `\n` 这种上古分割方式硬切你的人生。
- 会调字号、调每页行数，尽量让你少翻白眼。
- 会让你用 **HEX + RGB** 改文字颜色，想把字调成赛博蓝、毒液绿、深渊粉，都行。
- 会记住你配过的快捷键，不会今天装懂明天失忆。
- 会在状态栏、Dock、或两边一起出现，主打一个随叫随到。
- 会置顶，像个不讲武德但很敬业的桌面幽灵。
- 会记住窗口位置和大小，不会每次启动都假装第一次见你。

## 运行要求

- macOS 13+
- 当前构建：Apple Silicon

也就是说，如果你拿着一台比较新的 Mac，想认真读点东西，或者假装认真读点东西，它就能为你服务。

## 安装方式（认真版，别笑）

### 从 GitHub Release 把它请回家

1. 下载 `mmReader.dmg`
2. 双击打开 DMG
3. 把 `mmReader.app` 拖进 `Applications`
4. 如果第一次打开被 Gatekeeper 拦住：
   - 在 Finder 里右键 `mmReader.app`
   - 点 `Open`
   - 再确认一次

就是这么朴素。没有炼金阵，没有管理员献祭仪式，没有“重启三次再执行某条远古终端咒语”。

## 默认快捷键（以及你完全可以改掉它们）

`Controls` 里可改。

常见默认项：
- `⌘O`：打开文件
- `⌘B`：显隐工具栏
- 翻页键：取决于当前快捷键配置，不搞一刀切独裁

## 本地开发（开发者挨打区）

### 跑全量测试

```bash
swift test
```

当你想确认整个项目还没被自己搞塌时，用这个。

### 跑单个测试

```bash
swift test --filter appDelegateOpenDocumentReplacesDisplayedText
```

当你只想精准确认“Open 到底有没有把旧文本踹下去”时，用这个。

### 构建 app

```bash
bash scripts/build_app.sh
```

产出桌面可执行生物体：
- `build/mmReader.app`

### 构建 DMG

```bash
bash scripts/build_dmg.sh
```

产出可以发给别人、让别人也一起加入阅读邪教的安装包：
- `build/mmReader.dmg`

## 已知限制

- 当前包还没有 notarize。换句话说，系统不会立刻把它当成天选之子。
- 当前构建目标是 Apple Silicon。Intel 机器别硬刚，硬刚大概率伤感情。

## 总结

这不是一个“低调、内敛、安分守己”的阅读器。

这是一个：
- 会漂浮
- 会记仇（记窗口）
- 会变色
- 会换壳（状态栏 / Dock）
- 会记住你按过什么键
- 还会把 TXT 当成重要文献郑重对待的离谱工具

如果你觉得普通阅读器太普通，`mmReader` 很适合你。

如果你觉得它有点神经病。

那就对了。
